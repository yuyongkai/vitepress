介绍 #

该项目是基于vite2 + vue3 + typescript + element-plus 二次封装组件，提供了一下：

  • 伸缩菜单
  • 图标选择器
  • 省市区选择器
  • 日期选择器
  • 城市选择器
  • 趋势标记
  • 通知列表
  • 通知菜单
  • 导航菜单
  • 动态进度条
  • 可配置型表单
  • 可配置型表格
  • 弹出框表单
  • 日历

具体用法请参考一下文档说明，了解组件具体使用方式。
如需预览效果，请点击右上角github站点
